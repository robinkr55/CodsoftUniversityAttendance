# Codsoft University Attendance

An Android application built using Kotlin and Jetpack Compose.
This app helps manage and track university student attendance in a simple and organized way.

## Features
- Add students
- Mark attendance (Present / Absent)
- View attendance records
- Simple and clean UI

## Tech Stack
- Kotlin
- Jetpack Compose
- Android Studio
